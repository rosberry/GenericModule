# Generic Module


## Requirements

- swift 5.3
- iOS 12 or bigger

## Installation
### Carthage (preffered)
```
github rosberry/GenericModule
```

## Swift Package Manager
```
dependencies: [
    ...
    .package(url: "https://github.com/rosberry/GenericModule", from: "1.0.0"),
    ...
]
```
